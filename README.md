# Hogar San Blas

Este es el repositorio para la plataforma informativa del Hogar San Blas de Nicoya

## Configuracion opcional de imagenes

- `VITE_ENABLE_EXTERNAL_IMAGE_PROXY=false`: deshabilita el proxy de imagenes externas via `wsrv.nl` para URLs no Cloudinary.
- Valor por defecto: habilitado. Si falla la carga por proxy, cada imagen vuelve a su URL original como fallback.
