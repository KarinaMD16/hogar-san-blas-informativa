# Hogar San Blas

Este es el repositorio para la plataforma informativa del Hogar San Blas de Nicoya

## Configuracion opcional de imagenes

- `VITE_ENABLE_EXTERNAL_IMAGE_PROXY=true`: habilita el proxy de imagenes externas via `wsrv.nl` para URLs no Cloudinary.
- Valor por defecto: deshabilitado. Si no se define, las URLs no Cloudinary se sirven tal cual.
